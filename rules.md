# Project Context: Developer Blog

## 핵심 목표
- 이 블로그는 Cloud Infrastructure 및 DevOps 역량을 보여주는 포트폴리오 사이트임.
- **최우선 순위:** 구글 검색 엔진 최적화(SEO). 모든 코드 생성 시 SEO를 고려할 것.

## 기술 스택
- Framework: Astro (SSG)
- UI: Tailwind CSS (Dark Mode 우선)
- 배포: GitHub Pages (GitHub Actions 활용)

## 에이전트 수칙 (Always Remember)
1. **SEO 강박:** 새로운 컴포넌트나 페이지를 만들 때마다 `meta tag`, `alt` 속성, `semantic tag`가 누락되지 않았는지 체크할 것.
2. **이름 규칙:** 파일명은 소문자 케밥 케이스(kebab-case)를 사용함.
3. **불필요한 설명 생략:** 코드를 먼저 보여주고, 핵심적인 로직 설명만 덧붙일 것.
